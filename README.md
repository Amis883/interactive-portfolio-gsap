# ✨ Interactive Portfolio – Next.js + GSAP

A modern, interactive portfolio built with **Next.js, TypeScript, Tailwind CSS, and GSAP**, inspired by editorial-style design and motion-driven user experiences.

This project focuses on **smooth animations, clean UI, and performance**, reflecting real-world frontend engineering practices.

---

## 🚀 Live Demo

👉 https://your-domain.vercel.app/

---

## 🧠 Concept

Instead of a traditional static portfolio, this project introduces:

* **Card-based layout**
* **Interactive focus transitions**
* **GSAP-powered animations**
* **Dynamic content expansion**

When a user clicks on a card:

> it smoothly moves to the center and expands into a detailed view.

---

## 🛠️ Tech Stack

* **Framework:** Next.js (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Animations:** GSAP (GreenSock)
* **State Management:** React Hooks

---

## 📁 Project Structure

```
app/
 ├─ page.tsx        # Main entry
 ├─ layout.tsx      # Global layout

components/
 ├─ Card.tsx        # Individual card
 ├─ CardGrid.tsx    # Layout system
 ├─ ExpandedCard.tsx# Focused view

lib/
 ├─ data.ts         # Portfolio content
```



